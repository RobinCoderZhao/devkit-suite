## 一、产品定位

### 1.1 一句话定义

> **"SaaS 行业第一情报平台 — AI 驱动的行业日报 + 竞品变更监控"**

DevKit Suite 不是通用网页监控工具，而是专为 **SaaS 行业从业者** 打造的情报中枢，聚焦于用 AI 将海量信息转化为可执行的商业洞察。

### 1.2 目标市场

| 客群 | 规模 | 痛点 | 使用场景 |
|------|------|------|----------|
| **SaaS 创始人/CEO** | 小型 | 没时间逐一跟踪竞品 | 竞品改了定价需立刻响应 |
| **产品经理** | 中型 | 竞品上了新功能不知道 | 评估是否跟进竞品 feature |
| **市场/增长** | 中大型 | 行业变化太快跟不上 | 每天 3 分钟了解行业全貌 |
| **投资人/分析师** | 机构 | 赛道信息碎片化 | 判断竞争态势和行业趋势 |
| **AI 开发者/技术 Lead** | 小型 | API 变更导致线上故障 | 第一时间知道 API 变化 |

### 1.3 核心价值主张

```
竞品做了什么？→ WatchBot 告诉你
行业发生了什么？→ NewsBot 告诉你
这意味着什么？→ AI 分析告诉你
```

## 二、两大核心产品

### 2.1 📰 NewsBot — SaaS 行业 AI 日报

> "每天 3 分钟，掌握 SaaS 行业全部重要动态。"

**已实现能力：**

| 能力 | 状态 | 说明 |
|------|------|------|
| 8+ 数据源采集 | ✅ | HackerNews, TechCrunch, VentureBeat, OpenAI Blog 等 |
| LLM 智能摘要 | ✅ | Gemini/MiniMax 驱动，筛选 + 评分 + 摘要 |
| 6 语言翻译 | ✅ | 中/英/日/韩/德/西，一次分析 + N 次翻译 |
| 精美 Email 推送 | ✅ | 暗色主题 Newsletter，重要性徽章 + 标签 |
| 订阅者管理 | ✅ | CLI 管理，支持多语言偏好 |

**商业化待实现：**

| 能力 | 优先级 | 说明 |
|------|--------|------|
| Web 日报浏览 | P0 | 用户登录后查看历史日报（→ [前端设计 §3.3](./frontend_design.md#33-newsbot--日报列表)）|
| 关键词过滤 | P1 | Pro 用户自定义关注主题 |
| 更多数据源 | P1 | 扩展至 20+ 源，覆盖 SaaS 垂直行业 |
| 每周深度报告 | P2 | AI 生成周度趋势总结 PDF |

### 2.2 🔍 WatchBot — AI 竞品变更监控

> "你的竞品改了价格、砍了功能——我比你先知道。"

**已实现能力：**

| 能力 | 状态 | 说明 |
|------|------|------|
| 多竞品多页面监控 | ✅ | pricing / changelog / features 等 |
| 定时抓取 + 快照存储 | ✅ | Cron 驱动，SHA256 去重 |
| HTML diff 引擎 | ✅ | unified diff + 行统计 |
| Gemini LLM 深度分析 | ✅ | 语义提取变更含义、具体模型名/功能名 |
| 按竞品分组邮件报告 | ✅ | 竞品 header → 页面子项，重要性徽章 |
| 多模型支持 | ✅ | gemini-3-flash-preview / pro-preview |

**商业化待实现：**

| 能力 | 优先级 | 说明 |
|------|--------|------|
| Web Dashboard | P0 | 竞品管理 + 变更时间线（→ [前端设计 §3.4-3.6](./frontend_design.md#34-watchbot--竞品总览)）|
| 多通知渠道 | P0 | Webhook + Slack + 企业微信/飞书 |
| 变更历史浏览 | P1 | 时间线 + Diff 可视化查看器 |
| 用户自定义 URL | P1 | 用户自行添加监控页面 |
| AI 趋势报告 | P2 | 周报/月报：竞品动向汇总 |
| 截图对比 | P2 | Playwright 截图 + 视觉 diff |

## 三、竞品分析

### 3.1 市场格局

| 层级 | 代表产品 | 定价 | 特点 |
|------|----------|------|------|
| **企业 CI** | Crayon, Klue | $1K-100K/mo | 战术卡片, CRM 集成, 销售赋能 |
| **中端 SaaS** | Visualping, Fluxguard | $10-250/mo | AI 摘要, 视觉检测, 区域选择 |
| **轻量级** | Distill.io, Hexowatch | $0-80/mo | 浏览器插件, 本地+云混合 |
| **开源** | changedetection.io | 免费 | Docker 部署, 70+ 通知渠道 |
| **DevKit Suite** | — | ¥0-149/mo | **SaaS 行业专精 + AI 深度分析** |

### 3.2 差异化优势

| 维度 | 通用工具 (Visualping等) | DevKit Suite |
|------|----------------------|--------------|
| 分析深度 | "页面变了" + 截图 | **"GPT-5.2 发布 + 定价下调 30%"** — 语义提取 |
| 配置复杂度 | 手动逐个添加 URL | **预置行业竞品库**，一键订阅 |
| 信息覆盖 | 仅变更监控 | **日报 + 监控** 双引擎 |
| 中文支持 | 无/差 | **中文原生**，面向中国 SaaS 团队 |
| 价格 | $10-250/mo | **¥49-149/mo** 性价比高 |

### 3.3 可借鉴设计

| 竞品 | 亮点 | 我们的借鉴 |
|------|------|------------|
| **Visualping** | 极简注册：粘贴URL → 选区域 → 完成 | 新用户引导流 |
| **Distill.io** | CSS Selector 精准选区 + 条件触发 | 支持 CSS selector |
| **Crayon** | AI 重要性评分 + 战术卡片 | 完善评分体系（已有） |
| **changedetection.io** | 70+ 通知渠道 | 优先加 Webhook |

## 五、实施路线图

### Phase 1: 可用产品 (4 周)

> 核心目标：用户能自助注册、浏览日报、管理竞品

| 周 | 前端 | 后端 |
|----|------|------|
| W1 | Landing Page + 注册/登录 | Auth API (JWT + Google OAuth) |
| W2 | Dashboard 首页 + NewsBot 日报浏览 | NewsBot 日报查询 API |
| W3 | WatchBot 竞品管理 + 时间线 | WatchBot CRUD API + 时间线查询 |
| W4 | 设置页 + 通知偏好 | 通知偏好 API + Webhook 通道 |

### Phase 2: 差异化 + 付费 (4 周)

| 周 | 前端 | 后端 |
|----|------|------|
| W5 | 定价页 + Stripe Checkout | Stripe 集成 + 计划管理 |
| W6 | Diff 可视化查看器 | Diff 详情 API |
| W7 | 用户自定义 URL 添加 | 自定义页面监控逻辑 |
| W8 | 响应式适配 + 性能优化 | Slack/企业微信通知通道 |

### Phase 3: 增长 (4 周)

| 周 | 前端 | 后端 |
|----|------|------|
| W9 | 关键词过滤 UI + 团队空间 | 关键词匹配引擎 |
| W10 | AI 趋势报告页面 | 周报/月报生成 |
| W11 | 竞品模板库（一键订阅行业包） | 预置竞品数据 |
| W12 | SEO 优化 + 公开日报 Web 版 | 公开内容 API |

## 七、运营成本估算

| 项目 | MVP | 100 用户 | 1000 用户 |
|------|-----|---------|---------|
| VPS (Go 后端 + Cron) | ¥70/月 | ¥140/月 | ¥350/月 |
| Vercel (前端) | 免费 | 免费 | ¥140/月 |
| LLM API (Gemini) | ¥35/月 | ¥210/月 | ¥1,400/月 |
| Stripe 手续费 | 2.9%+30¢ | 2.9%+30¢ | 2.9%+30¢ |
| 域名 + CDN | ¥70/年 | ¥70/年 | ¥70/年 |
| **总计** | **~¥180/月** | **~¥460/月** | **~¥2,100/月** |

**盈亏平衡点：** 4 个 Starter (¥49) 用户 或 2 个 Pro (¥149) 用户即可覆盖 MVP 成本。
