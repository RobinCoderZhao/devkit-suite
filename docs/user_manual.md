# DevKit Suite 用户使用手册 📖

欢迎使用 DevKit Suite！这是一个全方位的 **SaaS 行业情报指挥中心**。在这里，您可以全天候追踪竞品定价策略、了解每日最新的 AI 前沿资讯，并通过敏锐的智能规则引擎，在市场发生异动时第一时间获得警报。

本手册将引导您从零开始，掌握系统的核心能力。

---

## 一、 账号注册与初始化配置 (Onboarding)

1. 访问 DevKit Suite 的 Web 首页（例如 `http://43.98.84.1` 或您的专属域名），点击页面上的 **"Get Started"**。
2. 在注册页面输入您的邮箱和密码建立账户。
3. **行业选择 (Aha Moment)**：注册成功后，您将进入引导流程。请勾选您当前所处的行业（例如：`AI Models & APIs` 或 `DevTools`）。
4. 系统将在数秒内，**自动为您配置 3 个该行业顶级的竞品作为默认追踪目标**（例如自动为您追踪 OpenAI 的定价页）。无需任何额外的手动输入，即刻开启情报工作！

---

## 二、 核心功能模块详解

### 2.1 战术控制台 (WatchBot Dashboard)

WatchBot 是本平台的核心引擎，为您自动监控竞品官网的点滴变化。

* **全局卡片 (Battlecards)**: 登录后的首屏，横向矩阵展示您追踪的所有竞品。这里用最直观的数据板向您汇报：竞品追踪了多少个页面？本周有没有高风险（High Severity）的异动改版？
* **深潜时间线 (Timeline Diff Viewer)**: 点击任意一个竞品的“View Timeline & Diffs”按钮，进入情报详情页。
  * **非技术友好**: 您不需要看懂枯燥的 HTML 源码差异。系统利用 **Gemini 大模型**，将长篇大论的网页变化提炼为一句话摘要（例如：“企业版订阅价格从 $20 上调至 $25”），并自动打上严重程度标签。

### 2.2 智能热点情报流 (NewsBot)

我们为您节省每天刷推特和看新闻的时间。

* **导航栏进入**: 点击左侧边栏的 **NewsBot** 模块。
* **全网收录**: 系统每天在后台自动汇总全球超过 20 大顶级科技媒体与官方博客（如 TechCrunch, OpenAI Blog, 机器之心）。
* **瀑布流阅读**: 通过去重引擎和 LLM 内容提炼，展现当天的精华摘要列表。

## 三、 高阶功能：账户与规则中心

如果您是免费用户 (Free Plan)，您最高可以追踪 **2 个竞品**。为了应对更为复杂的商业环境，我们为您提供了丰富的 Pro 版高阶能力。

### 3.1 订阅与计费升级 (Billing)

* 点击侧边栏的 **Settings** -> 选择 **Billing & Plan** 面板。
* 在这里，您可以查阅当前的订阅容量。点击 "Upgrade to Pro"，您将被安全地引导至 **Stripe 收银台** 进行支付绑定。
* 专业版 (Pro Plan) 支持您追踪高达 **100 个**竞品网站，彻底解放业务范围。

### 3.2 智能防打扰报警 (Smart Alerts)

**【Pro 版专属特权】** 真正有价值的情报，应该是“货找人”，而不是天天去刷页面。

1. 在 **Settings** 面板中，切换至 **Smart Alerts** 页签。
2. **设定触发条件 (Condition)**: 您可以自由配置规则。
    * 例如：当变化摘要中*包含特定关键词*（如 `"pricing"`, `"降价"`, `"重大发布"`）。
    * 例如：当大模型判定这次改变的**风险等级** `>= High` (高严重度) 时。
3. **设定动作 (Action)**: 命中规则后，系统将为您投递 **Email 邮件** 或是向贵公司的钉钉/飞书触发 **Webhook** 通知。

---

## 四、 最佳实践建议

* **克制追踪范围**：不要追踪竞品的整个官网，而是精准添加他们最容易暴露商业动作的单页（如 `/pricing`, `/enterprise`, `/changelog`）。
* **依靠规则引擎**：建立 2-3 条核心关键词的 Smart Alerts，让系统作为您最忠诚的 24 小时商业斥候。
* **团队共享情报**：如果捕获到了“竞品降价”的高价值 Diff 视图，您可以直接将该 Dashboard 截图和由于大模型总结的内容，直接发给销售团队进行打单话术的跟进。
